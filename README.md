# Super Bowl Data Analysis

## Overview

This project analyzes historical Super Bowl data to identify trends in team performance, television viewership, and halftime show popularity.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset Files

* super_bowls.csv
* tv_data.csv
* halftime_show_artists.csv

## Analysis Performed

* Team performance comparison
* Historical match statistics analysis
* TV viewership trend analysis
* Halftime show artist popularity analysis
* Visualization of historical patterns

## Outcome

Performed exploratory data analysis on historical Super Bowl datasets to identify audience engagement patterns and team performance trends.
